<h1>Eureka Server using Spring Boot </h1>

## Steps

- Clone as follows

```
git clone https://github.com/papicella/scs-eureka
```

- Package as shown below

```
$ ./mvnw -DskipTests package
``` 

- Run as follows

```
$ ./mvnw spring-boot:run
```

Invoke Eureka Server UI as follows

```
http://localhost:8761/
```
